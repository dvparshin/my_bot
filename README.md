# Телеграм бот

## Локальный запуск
1) Скачать текущий репозиторий  
```bash
git clone git@github.com:dvparshin/my_bot.git
```
2) Перейти в каталог репозитория  
```bash
cd my_bot
```
3) Установить библиотеку python-telegram-bot  
```bash
pip install -r requirements.txt
```
4) Запустить бот  
```bash
python3 bot.py
```
